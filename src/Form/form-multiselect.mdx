A text style input that allows the user to select from multiple options.

### Basic usage

#### Controlled Usage

```jsx live
() => {
    const options = [
        'White',
        'Black',
        'Green',
        'Purple',
        'Blue'
    ];
    const [selectedOptions, setSelectedOptions] = useState(['Purple', 'Green']);

    return (
        <Form.Multiselect
            floatingLabel="Colors"
            options={options}
            selectedOptions={selectedOptions}
            setSelectedOptions={setSelectedOptions}
        />
  );
}
```
