Form auto-suggest enables users to manually select or type to find matching options to partial text input.

## Controlled Standalone Usage

```jsx live
() => {
    const [data, setData] = useState({ title: '' });

    return (
        <Form.Autosuggest
            floatingLabel="This is floating label"
            helpMessage="This is help message"
            aria-label="form autosuggest"
            value={data.title}
            onChange={
                (value) => setData(prevState => ({ data: { ...prevState.data, org: value } }))
            }
        >
            <Form.AutosuggestOption>Green</Form.AutosuggestOption>
            <Form.AutosuggestOption>Red</Form.AutosuggestOption>
            <Form.AutosuggestOption>Blue</Form.AutosuggestOption>
            <Form.AutosuggestOption>Black</Form.AutosuggestOption>
        </Form.Autosuggest>
    );
}
```
