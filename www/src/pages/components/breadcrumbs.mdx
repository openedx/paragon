# Breadcrumbs

##### Basic Usage

```jsx live=true
<Breadcrumb
  links={[
    { label: 'Link 1', url: '/link-1' },
    { label: 'Link 2', url: '/link-2' },
    { label: 'Link 3', url: '/link-3' },
  ]}
/>
```

##### With active label

```jsx live=true
<Breadcrumb
  links={[
    { label: 'Link 1', url: '/link-1' },
    { label: 'Link 2', url: '/link-2' },
    { label: 'Link 3', url: '/link-3' },
  ]}
  activeLabel="Link 4"
/>
```

##### With custom spacer

```jsx live=true
<Breadcrumb
  links={[
    { label: 'Link 1', url: '/link-1' },
    { label: 'Link 2', url: '/link-2' },
    { label: 'Link 3', url: '/link-3' },
  ]}
  spacer={<span className="custom-spacer">/</span>}
/>
```

##### Props

<StaticQuery
  query={graphql`query {
    componentMetadata(displayName: { eq: "Breadcrumbs" }) { ...ComponentDocGenData }
  }`}
  render={({ componentMetadata }) => {
    if (componentMetadata == null) return null;
    return <PropsTable propMetaData={componentMetadata.props} />;
  }}
/>
